# Back End

- el back End esta construido en Node js.


## Ejecutar Back End

-En la carpeta raiz del projecto ir al directorio proyecto_sitio_videos.

### 'cd proyecto_sitio_videos'

- Instalar paquetes

### 'npn install'

- ejecutar aplicacion

### 'npm start'
